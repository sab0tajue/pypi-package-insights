Bulk supply-chain analytics for Python packages: download trends, version
cadence, dependency depth, license, **known vulnerabilities (OSV)**, and a
transparent **Risk Score 0–100**.

Pulls from PyPI, pypistats.org, Google deps.dev, and osv.dev in one call.
List mode (your requirements) or search mode (PyPI substring).

Use cases: security/OSPO scoring, DevOps gating, OSS due diligence.
